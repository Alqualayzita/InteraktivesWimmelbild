Interaktives Wimmelbild

Ein Gerichtsprozess für die Umwelt

Über Game

Download

Über den entsprechenden Downloadlink ( Windows / Mac ) kann das Spiel heruntergeladen werden. Auf Windows anschließend den Ordner entzippen und zum Spielen die Datei "InteraktivesWimmelbild.exe" starten. Auf Mac ebenfalls die Datei entzippen und zum folgendem Ordner navigieren: GameMac.app/Contents/MacOS und die Datei starten.

Sourcecode

Der Sourcecode kann hier heruntergeladen werden. Das Projekt kann folgendermaßen in Unity geöffnet werden: Den Ordner herunterladen und entpacken, im Unity Hub auf "Add" klicken, dann den Ordner auswählen und anschließend in der Projektübersicht auf "Interaktives Wimmelbild" klicken.
